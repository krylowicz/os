# Making my own os from scratch inspired by Nick Blundells [paper](https://www.cs.bham.ac.uk/~exr/lectures/opsys/10_11/lectures/os-dev.pdf)

## To run
  1. `make`
  2. `make run`

## Features:
  - [x] boot without grub
  - [x] 32-bit mode
  - [x] assembly to c jump
  - [ ] interrupt handling
  - [ ] screen output and keyboard input
  - [ ] memory management
  - [ ] filesystem
  - [ ] shell
